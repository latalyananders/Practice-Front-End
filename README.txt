Должен быть установлен:

	Node JS        https://nodejs.org/en/
	Git            https://git-scm.com/

Установка сборки:

	$ npm install -g gulp

	$ npm i
	$ npm install -g bower
	$ bower i

*если возникнут ошибки при установке bower, попробуйте через обычную консоль, а не bush

Запуск для разработки:

	$ gulp

Сборка проекта:

	$ gulp public



***

Вся разработка ведется в /app/

Gulp смотрит в /app/assets/ и на /app/*.html (в них не править ничего)

Картинки сразу в /assets/images класть

Собирается проект в /dist/ (там не править ничего)

***

Используются Pug для структуры и Scss для стилей
В pug/ лежат страницы(pages/) и отдельные блоки(templates/)
Каждый блок должен лежать там,а на главной странице только подключения,футер,хедер
Стили и скрипты,принадлежащие блоку, лежат в blocks/ в соответствующей папке
В config/ миксины и переменные
